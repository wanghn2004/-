### minio

web端口 8040

api端口 8030

桶路径 /data/minio

日志 /home/software/minio/minio.log

```bash
# nohup：忽略终端退出信号，保证服务后台持续运行（关闭终端不中断）
# ./minio server：启动 MinIO 服务端，后跟数据存储目录
# /data/minio：指定 MinIO 数据存储根目录（需提前创建并配置读写权限）
# --console-address ":8040"：自定义 Web 管理控制台端口为 8040（访问地址：http://服务器IP:8040）
# --address ":8030"：自定义 API 接口端口为 8030（用于客户端/程序对接存储服务）
# > /home/software/minio/minio.log：将服务标准输出日志写入指定文件
# 2>&1：将错误输出（stderr）重定向到标准输出（stdout），统一记录到日志文件
# &：将服务进程放入后台运行，不占用当前终端
nohup ./minio server /data/minio --console-address ":8040" --address ":8030" > /home/software/minio/minio.log 2>&1 &
```
